# Life RPG

Геймификация жизни: прокачка PM/Product/Дисциплины, форма, квесты спринта,
аватар, который меняется вместе с тобой. Vite + React + TypeScript + Tailwind.
Данные хранятся в браузере (localStorage) + экспорт/импорт JSON-сохранения.

## Локальный запуск

Нужен установленный Node.js (LTS, nodejs.org).

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`.

## Деплой на Vercel через GitHub

1. **Создай GitHub-аккаунт**, если его нет — github.com → Sign up.
2. **Создай пустой репозиторий** (например `life-rpg`), без README — он уже есть.
3. В папке проекта:
   ```bash
   git init
   git add .
   git commit -m "Life RPG: первая версия"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_НИК/life-rpg.git
   git push -u origin main
   ```
4. Зайди на **vercel.com** → Sign up через GitHub → **Add New → Project** →
   выбери репозиторий `life-rpg` → **Deploy**. Vite определится автоматически,
   ничего настраивать не нужно. Через минуту получишь ссылку вида
   `life-rpg.vercel.app`.
5. На телефоне открой ссылку в Safari/Chrome → **«Добавить на домашний экран»**.
   Получится иконка и полноэкранный запуск без адресной строки (PWA-манифест
   уже настроен в `public/manifest.webmanifest`).

Дальше — просто редактируешь код и пушишь (`git push`), Vercel передеплоит
автоматически.

## Что где лежит

- `src/engine/rules.ts` — вся игровая математика в одном месте: формулы XP,
  содержание навыков (upkeep), кривые формы, веса героя. Хочешь жёстче
  падение — крути коэффициенты здесь.
- `src/engine/engine.ts` — чистая логика (закрытие дня, начисление/списание
  XP, откат уровня, идле-дебафф). Без React, можно переиспользовать хоть в
  Telegram-боте.
- `src/engine/store.tsx` — React state + localStorage + экспорт/импорт.
- `src/components/` — экраны «Персонаж» и «Дашборд».
- `src/assets/avatar/street.png` и `office.png` — твой аватар (уличный тир /
  офисный тир, разблокируется на 25 уровне Product). Сейчас это временные
  заглушки — просто замени эти два файла своими картинками с тем же именем,
  ничего в коде менять не нужно.

## Как устроена механика (кратко)

- Все навыки и форма — шкала **1–100**.
- Уровень не только растёт, но и падает: у каждого навыка есть **содержание**
  (upkeep) — XP, которое списывается каждый день просто за то, что ты стоишь
  на месте. Чем выше уровень, тем дороже содержание. Не заработал достаточно
  — уровень откатывается. Есть дно: не ниже (пик − 15).
- **Активность за 7 дней** (дуга на экране персонажа) — сумма всего
  заработанного XP за неделю. Ниже 25/100 — включается дебафф «Простой»:
  содержание навыков растёт на 40%.
- **Форма** = 50% регулярность (XP тренировок за 14 дней) + 50% тело и
  результаты (подтягивания/отжимания/% жира по кривой). Вес отображается
  только как тренд, XP не начисляет и не забирает.
- **Заморозка** — тумблер на экране персонажа: содержание не списывается, но
  и XP не капает. Используй на время болезни/отпуска.
- **Квесты спринта** — живут 21 день, тапаешь «+»/«−» для прогресса, XP
  начисляется сразу.
- **Общий уровень героя** = 30% PM + 25% Product + 20% Форма + 15%
  Дисциплина + 10% Активность.

## Известное упрощение vs. исходная идея

В обсуждении в другом чате предполагалось, что вечерний отчёт можно
надиктовать свободным текстом, а Claude через API сам парсит его в XP. Это
требует бэкенда с твоим API-ключом Anthropic — в этой версии вместо этого
простые числовые поля и тапы по квестам (экран «Дашборд» → «Итог дня»),
без параллельного сервиса. Если захочешь версию с голосовым вводом и
AI-парсингом — это добавляется отдельной serverless-функцией на Vercel
(Edge Function), дай знать.

## Экспорт/импорт сохранения

На экране «Дашборд» внизу — «Скачать сохранение» (JSON) и «Загрузить».
Полезно, пока не подключена синхронизация между устройствами (например,
через Supabase — можно добавить позже).
